# BOJ_S3_ 11726_2xn타일링이2023.07.03] </br>

## 알고리즘 종류
```
다이나믹 프로그래밍(DP)
```

## 해설 및 느낀점
```
DP에서 이미 계산된 값을 또 계산 할 필요가 없기 때문에 memo라는 배열에 값을 저장해서 불필요한 계산을 줄였다.
그러나 마지막에 recursion을 하는 부분에서 오버플로우가 발생해서 계속 틀렸습니다가 떴다. 계산을 할 때 값을 초과할 수도 있으므로 값을 미리 나눠줘야 한다.
문제에서 나눈 값을 출력하라고 할 때 조심하면 될거 같다.
```
## 코드
```c++

#include <iostream>
#define MAXNUM 1001
#define NUM 10007
int memo[MAXNUM];
using namespace std;

int dp(int k)
{
    if(memo[k]!=0)
        return memo[k];
    if(k<2)
    {
        memo[k]=1;
        return 1;
    }
    else
        memo[k]=(dp(k-1)%NUM+dp(k-2)%NUM)%NUM;
    return memo[k];
}
int main()
{
    int n;
    cin>>n;
    cout<<dp(n)<<'\n';
    return 0;
}
```
